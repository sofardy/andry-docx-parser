# Описание скрипта

Этот скрипт предназначен для рекурсивного поиска файлов формата `.docx` в папке `files` и её подпапках. Он извлекает из файлов следующую информацию:

1. **Имя файла**
2. **Имя клиента** (парсится из названия файла)
3. **Email** (извлекается из содержимого файла)
4. **Номер телефона** (парсится из названия файла)
5. **Папка**, в которой находится файл (относительно папки `files`)
6. **Год** (текущий год)

## Как использовать

1. Убедитесь, что у вас установлен Python версии 3.6 или выше.
2. Установите зависимости, выполнив команду:
   ```bash
   pip install python-docx
   ```
3. Поместите `.docx` файлы в папку `files` или её подпапки.
4. Запустите скрипт:
   ```bash
   python index.py
   ```
5. После выполнения скрипта в корне проекта появится файл `output.csv`, содержащий собранные данные.

## Формат выходного файла

Файл `output.csv` содержит следующие столбцы:

- `file name` — имя файла
- `name` — имя клиента
- `email` — email клиента
- `phone` — номер телефона клиента
- `folder` — папка, в которой находится файл
- `year` — текущий год

## Пример структуры папок

```
files/
├── folder1/
│   ├── client1_123-456-7890.docx
│   └── client2_987-654-3210.docx
├── folder2/
│   └── client3_555-555-5555.docx
└── client4_444-444-4444.docx
```

## Пример выходного файла

```
file name,name,email,phone,folder,year
client1_123-456-7890.docx,Client 1,client1@example.com,123-456-7890,folder1,2025
client2_987-654-3210.docx,Client 2,client2@example.com,987-654-3210,folder1,2025
client3_555-555-5555.docx,Client 3,client3@example.com,555-555-5555,folder2,2025
client4_444-444-4444.docx,Client 4,client4@example.com,444-444-4444,.,2025
```
